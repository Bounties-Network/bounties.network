# bounties.network

The website project for https://bounties.network

## Development

```sh
# clone
git clone https://github.com/Bounties-Network/bounties.network.git
cd bounties.network

# install dependencies
npm install

# development watching for changes
npm run dev

# build and compact for production
npm run prd
```
