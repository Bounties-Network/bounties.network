Solve for p

p = T / X is the trivial solution. For the complex solution, see proof.

Proof.
m/(n+1)(x-X+T/p)^n+1 - m/(n+1)*x^(n+1) = T - Xp.
We multiply by p^(n+1).
m/(n+1)(x*p + (T-X*p))^(n+1)-m/(n+1)*(x*p)^(n+1) = (T-X*p)*p^(n+1).
Now, expanding the former polynomial on the left, the first term will cancel with the latter polynomial, so we have:

m/(n+1)*[(x*p)^n(T-X*p) + (x*p)^(n-1)(T-X*p)^2 + ... + (T-X*p)^(n+1)] = (T-X*p)*p^(n+1).
We would divide by (T-Xp), but then we must consider our first solution: p = T / X. We verify the solution in the main equation and it is correct. Moving forward, we have:

m/(n+1)[(x*p)^n + (x*p)^(n-1)(T-X*p)+...+(T-X*p)^n]=p^(n+1).
But then, we see on the left side of the equation an expanded polynomial, which we implode back:

m/(n+1)(x*p+T-X*p)^n=p^(n+1).
This is:

m/(n+1)[(x-X)*p+T]^n=p^(n+1).
We note this (*). Now, we expand again the left side polynomial:

m/(n+1)[(x-X)^n*p^n + (x-X)^(n-1)*p^(n-1)*T+...+T^n]=p^(n+1).
Moving terms here and there, we get:

-p^(n+1)+m/(n+1)(x-X)^n*p^n+m/(n+1)(x-X^(n-1)*T*p^(n-1)+...+m/(n+1)(x-X)T^(n-1)*p=-m/(n+1)*T^n.
But then,

-p[p^n + (X-x)*m/(n+1)*(x-X)^(n-1)*p^(n-1)+m/(n+1)(X-x)(x-X)^(n-2)*p^(n-2)*T+...+(X-x)m/(n+1)*T^(n-1)] = -m/(n+1)*T^n.
Again, imploding the polynomial we have:

-p{p^n+m/(n+1)(X-x)[(x-X)*p+T]^(n-1)}=-m/(n+1)*T^n.
In fact, since p=T/(X-x) is not a solution, then:

m/(n+1)[1/(1+T/((x-X)*p)][(x-X)*p+T]^(n-1)=-m/(n+1)T^n+p^(n+1).

Remembering (*), we have:

p^(n+1)/[1+T/((x-X)*p)] = -m/(n+1)T^n+p^(n+1).
Dividing by p^(n+1) (not zero from initial conditions), we have:

1/(1+T/((x-X)*p))=-m/(n+1)*T^n/(p^(n+1)) + 1.
Hence,

m/(n+1)*(T/p)^n*(1/p)-T/(p*(x-X)+m/(n+1)*(T/p)^(n+1)*1/(p*(x-X)) = 0.
We multiply by p^(n+2), having:

m/(n+1)*T*p-T/(x-X)*p^(n+1)+m/(n+1)T^(n+1)/(x-X) = 0.
Rearranging,

T/(x-X)*p^(n+1)-m/(n+1)*T*p+m/(n+1)*T^(n+1)/(x-X) = 0.
As we can see, the answer will vary very much in terms of n (this is a polynomial in p of grade n+1). You can know the solution once you know n.
